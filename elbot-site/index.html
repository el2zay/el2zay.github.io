Concrètement, Elbot c'est quoi?

Elbot est un bot discord créer et codé par el2zay et codé aussi à l'aide de Johan.
Il a des fonctions basique comme des commandes clear,say et infoserver. Il permet de diffuser de la musique dans des salons vocaux. Elbot est un bot principalement fun, il a plusieurs références française et il a plein d'autres fonctionnalités. Bien entendu Elbot est toujours en cours de développement et j'ajoute petit à petit des fonctions chaque semaines.
Pour voir toutes les commandes vous pouvez voir la page d'aide

  • En ligne 7j/7 24h/24
  • Gratuit et Open Source
  • Codé en JavaScript
  • Musique

Page d'aide

Ici vous trouverez toutes les commandes de Elbot. (le prefix est "e!")

COMMANDE ÉCRITE

pessi

Description : Pour connaitre tous les mots de pessis
Permission : Envoyer un message
WARN : Aucun

say

Description : Pour me faire dire tout et n'importe quoi
Permission : Gérer les messages
WARN : Vous ne pouvez pas mentionner @here/@everyone

uno

Description : Pour avoir la carte changement de sens
Permission : Envoyer une image et un message.
WARN : Aucun

sondage

Description : Crée un sondage avec les choix "oui" et "non" grâce aux réactions.
Permission : Envoyer un message et des réactions
WARN : Aucun

test

Description : Pour savoir si je fonctionne et lequel des codes fonctionnent.
Permission : Envoyer un message
WARN : Aucun

infoserver

Description : Pour connaitre les informations importantes sur ce serveur
Permission : Envoyer un message embed
WARN: Aucun

restart

Description : Redémarre / déconnecte et reconnecte le robot de Discord.
Permission : Créateur
WARNAucun

number

Description : Pour avoir un nombre au hasard entre 1 et 100.
Permission : Envoyer un message
WARN : Aucun

heberger

Description : Pour savoir sur quel hébergeur je suis héberger en ce moment!
Permission : Envoyer un message
WARN : Peut ne pas être à jour automatiquement

ping

Description : Affiche le temps de latence du bot en ms.
Permission : Envoyer un message
WARN : Aucun

clear

Description : Pour clear le nombre de message que vous souhaitiez
Permission : Gérer les messages
WARN : Max: 100

invite

Description : Donne un lien d'invitation pour inviter le bot sur votre serveur.
Permission : Aucune
Lieu : MP / Textuel

github

Description : Pour afficher le code github
Permission : Envoyer un message
WARN : Il est disponible sur ce site. Et le repo n'est pas mis à jour manuellement

D'autres commandes...

Une idée de commande ? Envoyez un message sur la fiche ou sur un de mes réseaux sociaux que vous trouverez sur la page principale Vous pouvez regarder aussi les prochaines commandes de Elbot.

COMMANDE MUSIQUE (LA MUSIQUE EST EN CE MOMENT HS MAIS REVIENT BIENTOT)

play

Description : Joue de la musique grâce à un lien youtube
Permission : Se connecter à un salon vocal et parler.
Exemple : e!play https://www.https//www.youtube.com/watch?v=n3UBSa7-H0U pour diffuser la musique Petrouchka dans un salon

stop

Description : Arrête la musique.
Permission : Se connecter à un salon vocal et parler.
WARN : Vous devez être dans le même salon que elbot pour arrêter la musique.

brique

Description : Pour que elbot chante tutititutu
Permission : Se connecter à un salon vocal et parler.
WARN : Disponible uniquement dans le salon #817012298057121852
