Concrètement, Elbot c'est quoi?

Elbot est un bot discord créé et codé par el2zay et codé aussi avec l'aide de Johan.
Il a des fonctions basiques comme des commandes clear,say et infoserver. Il permet de diffuser de la musique dans des salons vocaux. Elbot est un bot principalement fun, il a plusieurs références françaises et il a plein d'autres fonctionnalités. Bien entendu Elbot est toujours en cours de développement et j'ajoute petit à petit des fonctions chaque semaine.
Pour voir toutes les commandes vous pouvez voir la page d'aide

  • En ligne 7j/7 24h/24
  • Gratuit et Open Source
  • Codé 100% en Python
  • Musique
  • Surveillé par Elwatch
  • Database

COMMANDES ÉCRITES

prefix <newprefix>

Description : Changer le prefix du bot
Permission : Administrateur.

say <texte>

Description : Pour me faire dire tout et n'importe quoi
Permission : Gérer les messages
WARN : Vous ne pouvez pas mentionner @here/@everyone

chinese <texte>

Description : Pour me faire dire tout et n'importe quoi mais où les caractères sont transformés en caractères chinois.
Permission : Gérer les messages
WARN : Vous ne pouvez pas mentionner @here/@everyone
Exemple: e!chinesse Bonjour Elbot répond:
书口冂丿口凵尺

embed <titre>§<description>§
<footer>§<couleur>

Description : Créer un embed à partir de elbot VERSION BETA
Permission : Aucune
WARN : La séparation est §
Exemple: e!embed Ça c'est le titre§ça c'est la description§ça c'est le footer§et là, la couleur. (voir commande list_color)
